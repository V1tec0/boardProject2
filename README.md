# Это мой диплом

Здесь расположены файлы моего диплома:
- tableProject: API для функционирования
- tableFront: табло
- tableAdmin: админ. панель

Краткое описание: Тема моего диплома посвящена разработке информационного табло для учебного заведения

-------
## Для чего он вообще создавался?

Данный проект хоть и является дипломным, но его я писал не только для диплома, а чтобы, возможно когда-то в будущем, когда я доработаю его, начать продавать как полноценный продукт, ибо решение может быть вполне конкурентоспособным, а ситуации, которые закрывает данная система, некоторые платные системы либо могут только в крайне дорогих опциях покрыть это, либо вообще не могут. с другой стороны, большинство популярных решений подобного типа:
1. Делаются реальной командой разработчиков, со своими аналитиками, разрабами, тестировщиками, а не одним студентом в связке с ИИ, который делает почти всю аналитическую работу, а тебе лишь остается писать код
